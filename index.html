Feb.

Course

Udemy: Build Responsive Real World Websites with HTML and CSS

Course Final Project

Hello, world!

Although I had dabbled in HTML/CSS prior, this course was my first structured learning environment on the subject. Despite it teaching some now-outdated practices (looking at you, float layout), it's an excellent way to hit the ground running with the fundamental concepts of building a webpage.

March

Course

Udemy: Web Design for Web Developers: Build Beautiful Websites!

Course

A Crash Course on Web Design Best Practices

This was the first of many resources I plan to use in a quest to fill what I believe is the biggest gap in my dev game - web design. It's a brief, yet loaded course that will leave you with some practical tidbits to apply to your next webpage.

April

Course

Udemy: Advanced CSS and Sass: Flexbox, Grid, Animations and More!

Course Starter Project Flexbox Project CSS Grid Project

CSS Master

A solid course for learning modern layout techniques, SASS/SCSS, and some crafty CSS tricks to keep in your pocket (that checkbox hack is a doozy!). The advent of Flexbox and Grid are gamechangers for CSS and for devs everywhere, and this is an extremely helpful resource in learning their use cases and capabilities.

May

Project

Freelance: JA Dancik Electric

Website

My First Freelance Project

A simple local business site built with pure HTML, CSS/SCSS, and Javascript - no frameworks or bloat, resulting in a well-performing site. Hosted on Netlify via GitHub repo with form submissions handled by Netlify.

June

Course

Harvard CS50: Introduction to Computer Science

Course

A True Intro to Programming

This course was rigorous yet rewarding. To be candid, I didn't finish completely and instead moved onto FreeCodeCamp once I grasped the core concepts. The course teaches mostly C, with some Python and SQL sprinkled in. Very entertaining lectures and challenging problem sets. Be patient with this one!

July

Book

Refactoring UI

Book

Web Design Made Practical

A web developer's design bible, if you will. In ~200 pages, this e-book teaches you web design in a practical way. It touches on concepts you may not have thought of, like creating depth, designing systematically, and so much more. Highly recommend for anyone looking to up their web-design game!
